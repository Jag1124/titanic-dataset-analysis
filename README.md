# Titanic Data Analysis

This project is based on Titanic dataset analysis using Python.

In this project, the dataset was cleaned, missing values were handled, and new features were created.  
Different types of analysis were performed to understand survival patterns.  
Visualizations like histogram, heatmap, and bar plots were created using matplotlib and seaborn.

Tools used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

This task is part of data analysis practice.
